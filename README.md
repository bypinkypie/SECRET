# Carbon Tracker

Aplikasi pelacak emisi karbon berbasis GPS.

## 🚀 Deploy ke Firebase Hosting

```bash
npm install -g firebase-tools
firebase login
# Edit .firebaserc — ganti project ID dengan punyamu
firebase deploy
```

## 📦 Push ke GitHub

```bash
git init
git add .
git commit -m "initialize"
git branch -M main
git remote add origin https://github.com/bypingkypie/SECRET.git
git push -u origin main
```
